mybatis逆向工程总结

发表于 2019-03-16 | 分类于 框架 |

概述

  • mybatis逆向工程可以针对单表自动生成mybatis执行所需要的代码（pojo、mapper.java、mapper.xml）。
  • 实际开发中，常用的逆向工程方式：由数据库的表生成java代码。

注意事项

mapper.xml文件已存在时，如果进行重新生成则mapper.xml文件内容不被覆盖而是进行内容增加，结果会导致mybatis解析失败。
解决办法：删除原来的已经生成的mapper.xml文件重新生成。
mybais自动生成的pojo以及mapper.java文件不是内容追加而是直接覆盖没有问题。
