概述

n 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

题目描述
给定一个整数 n，返回所有不同的 n 皇后问题的解决方案。
每一种解法包含一个明确的 n 皇后问题的棋子放置方案，该方案中 ‘Q’ 和 ‘.’ 分别代表了皇后和空位。

示例：

算法思路

采用回溯算法，深度优先遍历所有可能，输出满足要求的可能。
表示方式： arr[i] = k; 表示： 第i行的第k个位置放一个皇后。
算法步骤：

  1. 从第一行第一个位置开始遍历，先判断是否满足要求，若满足，则放置一个皇后；
  2. 然后递归调用下一行；
  3. 若所有行都被考虑过，则得出一种解法；
